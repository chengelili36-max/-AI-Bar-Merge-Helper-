# 🍸 时尚百货城：酒吧合并小游戏 AI 辅助器

这是一款专门为“时尚百货城”内置小游戏设计的**合并策略辅助工具**。当关卡难度提升、空间拥挤时，本工具通过 AI 算法计算最佳落点，帮助你实现连击消除，快速通关。

---

## 🚀 核心功能

- **1:1 模拟棋盘**：完全还原 4x5 布局，支持点击编辑任意格子状态。
- **复合托盘支持**：完美适配游戏中的**单格**与**双格（长条）**托盘。
- **动态难度调节**：支持 2~12 种酒水配置，匹配玩家当前的解锁进度。
- **AI 深度启发式搜索**：接入 Claude 3.5 模型，不仅考虑当前合并，更预测合并后的空格释放。
- **视觉预警**：
  - 🟢 **绿色**：满 6 杯，可立即消除。
  - 🟠 **橙色**：4-5 杯，高价值合并目标。
  - 🔵 **虚线**：空格子，建议落点。

---

## 🕹️ 游戏规则适配说明

本助手内置了精准的物理引擎逻辑：
1. **合并优先级**：放置后按“中 -> 上 -> 下 -> 左 -> 右”顺序检测。
2. **强制约束**：
   - 托盘必须放置在 **0/6** 的空位中。
   - 双格托盘不可拆分，必须占用垂直相邻的两个空格。
   - 3个托盘必须分布在互不重叠的坐标上。

## 📖 使用指南

### 1. 初始化设置
在页面顶部选择当前关卡**已解锁的酒水种类数量**。这会重置棋盘以匹配当前游戏的色池。

### 2. 同步游戏状态
观察手机游戏画面，点击辅助器对应的格子，填入现有的酒水种类（点击“+”或“—”）。

### 3. 编辑吧台托盘
点击底部的托盘卡片，选择托盘形状（单/双），并输入托盘上待放置的酒水。

### 4. 获取最优解
点击 **[🤖 获取最优摆放建议]**。
- AI 会给出具体的坐标建议，例如：`托盘1 -> [2, 3]`。
- 附带详细的逻辑分析，告诉你为什么要这样放（例如：能触发哪几种颜色的连消）。
